# 📄 AI-Powered Resume Screening and Application Screening System

## 📌 Project Overview

The **AI Resume Screening System** is a web-based application that automates the process of screening resumes by comparing candidate resumes with a job description. The system extracts relevant skills, calculates an ATS (Applicant Tracking System) score, identifies matched and missing skills, and helps recruiters evaluate candidates more efficiently.

---

## 🚀 Live Demo

🔗 **Streamlit Application:**

## ✨ Features

- 📄 Resume Upload (.docx)
- 📝 Job Description Input
- 🔍 Resume Parsing
- 🧠 Skill Extraction
- 📊 ATS Score Calculation
- ✅ Matched Skills Identification
- ❌ Missing Skills Analysis
- 📈 Candidate Ranking (Google Colab Version)
- 📁 CSV Report Generation
